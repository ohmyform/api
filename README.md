## Description

[OhMyForm](https://github.com/ohmyform) api backend

All calls to the api are through GraphQL, with the endpoint 
providing an introspectable schema at `GET /graphql`

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
